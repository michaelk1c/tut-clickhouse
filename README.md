# Clickhouse

## 1. Install

### 1. Use docker compose

- see the compose file in onec-devops3

### 2. Install client

#### 2.1 Mac

```
brew install clickhouse
```

## 2. Using client

1. set up .env file with direnv
2. use config file
3. call client with -c option to config file

```
clickhouse client -c config.xml
```
